Homebrew - my homebrew receipes

This project contains my homebrew receipes. To use the receipes, use brew tap tamada/brew.

  • rrh
    • Git repository manager.
  • uniq2
    • Extended uniq.
  • heatman
    • Creating heat map from given csv file.
  • qrg
    • QR code generator.
  • 9rules (ninerules)
    • Checking tool of small object programming for the Java language.
  • ttt
    • Checking diploma of courses in ISE, KSU.
  • pochi
    • Detecting the software theft, the birthmark toolkit for the JVM platform.
$ brew tap tamada/brew # <- at the first time
$ brew install rrh
$ brew install uniq2
$ brew install heatman
$ brew install qrg
$ brew install ninerules # <- 9rules
$ brew install ttt
$ brew install pochi

Project link: https://tamada.github.io/brew
