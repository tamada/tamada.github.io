tjdoe

December 26, 2019

🗣 Overview

tjdoe anonymizes given programs for programming courses and their scores for grades. For this, tjdoe converts names and ids appearing in the programs into anonymized ids.

tjdoe is abbreviation of “to john/jane doe.”

🍺 Install

$ brew tap tamada/brew # <- the first time only.
$ brew install tjdoe
