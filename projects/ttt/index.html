ttt

October 6, 2019

🗣 Overview

このツールは，KSU ISEにおける各コースの終了要件を単位取得科目から確認するためのツールです． ttt は「単位をたくさん取ろう」の頭文字です．

なお，このツールを使った事による不利益は，いかなる場合であっても一切保証しません．

🍺 Install

$ brew tap tamada/brew # <- the first time only.
$ brew install ttt
